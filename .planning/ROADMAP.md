# Roadmap - PipeLink CRM

## Milestone 1: Foundation & Core UI
- [ ] **Phase 1: Environment Setup**
  - Initialize Next.js project.
  - Setup Supabase project (Auth + DB).
  - Define Database Schema (Clients, Deals, Tags, Follow-ups).
- [ ] **Phase 2: Authentication & Layout**
  - Implement Login/Signup.
  - Create Premium Dashboard Layout (Sidebar, Header, Theme).
- [ ] **Phase 3: Client Management**
  - CRUD for Clients.
  - CRUD for Tags.

## Milestone 2: Kanban & Automation
- [ ] **Phase 4: Visual Kanban Board**
  - Build the drag-and-drop board.
  - Sync board state with Supabase.
- [ ] **Phase 5: n8n Integration**
  - Setup webhook triggers for follow-ups.
  - Create n8n workflows for SMS/Email.
- [ ] **Phase 6: Scheduling & Follow-ups**
  - Implement the calendar view.
  - Date-based task management.

## Milestone 3: Advanced Features & Polish
- [ ] **Phase 7: Document & History Management**
  - File uploads for budgets/receipts.
  - Detailed negotiation history timeline.
- [ ] **Phase 8: Analytics & Dashboards**
  - Financial summaries.
  - Funnel conversion stats.
- [ ] **Phase 9: Final Polish & UX**
  - Micro-animations.
  - Error handling and loading states.
  - Mobile responsiveness.
