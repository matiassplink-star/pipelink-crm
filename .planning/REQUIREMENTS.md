# Requirements - PipeLink CRM

## 1. Authentication & Security
- [ ] User registration and login via Supabase Auth.
- [ ] Role-based access control (Admin/User).
- [ ] Secure file storage for attachments in Supabase Buckets.

## 2. Kanban & Funnel Management
- [ ] Dynamic Kanban board with customizable columns (Stages).
- [ ] Drag-and-drop cards representing Leads/Deals.
- [ ] Lead status transitions and movement tracking.
- [ ] Tags system for categorization (e.g., "Hot Lead", "Follow-up", "VIP").

## 3. Client Management
- [ ] Detailed client registration form.
- [ ] Client profile page with:
    - Contact Info (Phone, Email).
    - Interaction Timeline (History).
    - Purchase/Negotiation History.
    - Attached Documents (Budgets/Receipts).
    - Custom Tags.

## 4. Automation & Follow-up
- [ ] Integration with n8n Webhooks.
- [ ] Automated message triggering for:
    - New client registration.
    - Funnel stage changes.
    - Scheduled follow-up dates.
- [ ] Multi-channel support: SMS and Email (extensible to WhatsApp).

## 5. Calendar & Scheduling
- [ ] Dashboard calendar view.
- [ ] Ability to schedule follow-up tasks directly from the client profile.
- [ ] Reminders/Notifications for upcoming tasks.

## 6. Financials & History
- [ ] Transaction log for each client.
- [ ] Budget/Proposal storage and tracking.
- [ ] Payment receipt uploads.

## 7. UX/UI & Aesthetics
- [ ] Cyber-Premium Dark Theme.
- [ ] Glassmorphic UI elements.
- [ ] Framer Motion for smooth Kanban card animations.
- [ ] Responsive design for desktop and tablet.
