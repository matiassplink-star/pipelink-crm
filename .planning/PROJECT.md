# PipeLink - Premium CRM with Kanban & Automation

## 🎯 Vision
PipeLink is a high-end, premium CRM designed for seamless client management, funnel optimization, and automated follow-ups. It integrates deeply with Supabase for data integrity and n8n for powerful communication workflows (SMS, Email, WhatsApp).

## 🚀 Core Features
- **Visual Kanban Board**: Drag-and-drop funnel management with premium aesthetics.
- **Complete Client Profiles**: Comprehensive registration with tags, history, and custom fields.
- **Smart Scheduling**: Integrated calendar for follow-ups and appointments.
- **Automation Engine**: Integration with n8n for triggering multi-channel messages (SMS, Email).
- **Document Management**: Attachment of budgets, receipts, and contracts.
- **Financial Intelligence**: Full purchase and negotiation history tracking.

## 🛠️ Tech Stack
- **Frontend**: Next.js (App Router), Vanilla CSS (Premium Custom Styles), Lucide Icons, Framer Motion.
- **Backend**: Supabase (Auth, Database, Storage, Edge Functions).
- **Automation**: n8n (External Integration).
- **Messaging**: Integration with SMS (Twilio/Vonage) and Email (Resend/SendGrid) via n8n.

## 🎨 Design Principles (Cyber-Premium)
- **Aesthetic**: Dark mode by default, glassmorphism, vibrant accent colors.
- **Interactions**: Micro-animations for state changes, smooth transitions between funnel stages.
- **Typography**: Modern sans-serif (Inter/Outfit).
